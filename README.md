# IHA KIRALAMA
Proje bir IHA kiralama uygulamasıdır. Authentication olmuş kullanıcılar mevcut IHAlardan kiralama yapabilirler.
Kiraladıklarını düzenleyebilir ve silebilirler.

## 1. Kullanılan Teknolojiler
- Django
- Django Rest Framework
- DataTable (server side)
- Swagger
- Postgresql
- Ajax
- JQuery
- Jazzmin
- Docker


## 2. Docker Konfigurasyonları
```bash
docker-compose build
```
```bash
docker-compose run django python manage.py migrate
```
```bash
docker-compose run django python manage.py createsuperuser
```
```bash
docker-compose up
```

Bu noktadan sonra proje 127.0.0.1:8000 adresinde çalışıyor!
